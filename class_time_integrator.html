TimeIntegrator::TimeIntegrator ( OdeArgument &  solver)

Constructor for the TimeIntegrator class. The Solver class is required to have a Solver.solve(Vector<double> &dst, const Vector<double> &src) method that will be called by the time integrator to find out about the solution to a given src.
